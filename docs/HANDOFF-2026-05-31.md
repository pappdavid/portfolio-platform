## Accessibility & Usability

### P1
- **Lift muted-gray contrast**: Status bar labels, column headers, meta table labels
  are likely below WCAG AA 4.5:1. Audit with DevTools and lift grays.

- **Respect prefers-reduced-motion**: The Typewriter component (landing-content.tsx
  lines 6-41) uses setInterval with no motion preference check.

### P2
- **Visible focus states**: Ensure buttons, links, nav tabs have visible focus outlines.

- **Mobile overflow at 360-390px**: Verify terminal window chrome + project table
  don't overflow horizontally on narrow phones.
